# Python 概述与安装

## 什么是 Python？

Python 是一种高级、解释型的编程语言，由 Guido van Rossum 在 1991 年首次发布。Python 的设计哲学强调代码的可读性和简洁的语法，特别是使用空格缩进来表示代码块。

### 核心特点

1. **简洁易读**：Python 的语法清晰，代码可读性强，适合快速开发
2. **功能强大**：拥有丰富的标准库和庞大的第三方库生态系统
3. **跨平台**：可在 Windows、Linux、macOS 等多种操作系统上运行
4. **面向对象**：完全支持面向对象编程，也支持函数式编程
5. **动态类型**：无需声明变量类型，解释器会自动推断
6. **解释型语言**：代码直接运行，无需编译步骤
7. **开源免费**：Python 是开源的，可以自由使用和分发

### Python 的应用领域

- **Web 开发**：Django、Flask 等框架
- **数据科学**：NumPy、Pandas、Matplotlib 等库
- **机器学习**：TensorFlow、PyTorch、Scikit-learn 等
- **自动化脚本**：系统管理、文件处理等
- **游戏开发**：Pygame 等库
- **网络编程**：爬虫、API 开发等

## Python 版本

### Python 2 vs Python 3

Python 2 已于 2020 年 1 月 1 日停止维护，现在应该使用 **Python 3**。

**主要区别**：
- Python 3 的 `print` 是函数，需要括号：`print("Hello")`
- Python 3 的字符串默认是 Unicode
- Python 3 的整数除法返回浮点数（使用 `//` 进行整数除法）
- Python 3 改进了异常处理语法

**本指南基于 Python 3.x 版本**。

### 当前版本

建议使用 Python 3.8 或更高版本，以获得更好的性能和功能支持。

## 安装 Python

### Windows 系统

1. 访问 [Python 官网](https://www.python.org/downloads/)
2. 下载最新版本的 Python 3.x 安装程序
3. 运行安装程序，**勾选 "Add Python to PATH"**
4. 选择 "Install Now" 完成安装

**验证安装**：
```bash
python --version
```

### macOS 系统

**方法一：使用官方安装程序**
1. 访问 [Python 官网](https://www.python.org/downloads/)
2. 下载 macOS 安装程序
3. 运行安装程序完成安装

**方法二：使用 Homebrew**
```bash
brew install python3
```

**验证安装**：
```bash
python3 --version
```

### Linux 系统

大多数 Linux 发行版已预装 Python，如果没有，可以使用包管理器安装：

**Ubuntu/Debian**：
```bash
sudo apt-get update
sudo apt-get install python3
```

**CentOS/RHEL**：
```bash
sudo yum install python3
```

**验证安装**：
```bash
python3 --version
```

## 环境配置

### 使用命令行

安装完成后，可以在命令行（终端）中运行 Python：

**Windows**：
```bash
python
```

**macOS/Linux**：
```bash
python3
```

这会启动 Python 交互式解释器（REPL），可以直接输入代码并查看结果。

### 退出 Python 解释器

在交互式解释器中输入以下任一命令退出：
- `exit()`
- `quit()`
- 按 `Ctrl+Z`（Windows）或 `Ctrl+D`（macOS/Linux）

### 运行 Python 文件

创建 `.py` 文件后，可以通过命令行运行：

**Windows**：
```bash
python script.py
```

**macOS/Linux**：
```bash
python3 script.py
```

## 第一个 Python 程序

### Hello, World!

创建一个名为 `hello.py` 的文件：

```python
# 这是我的第一个 Python 程序
print("Hello, World!")
```

运行程序：
```bash
python hello.py
```

输出：
```
Hello, World!
```

### 交互式编程示例

启动 Python 解释器，尝试以下代码：

```python
# 计算
>>> 2 + 3
5

>>> 10 * 5
50

# 字符串操作
>>> "Hello" + " " + "World"
'Hello World'

>>> len("Python")
6

# 变量赋值
>>> name = "Python"
>>> print(f"Hello, {name}!")
Hello, Python!
```

## 开发工具推荐

### 代码编辑器

- **VS Code**：免费，功能强大，插件丰富
- **PyCharm**：JetBrains 出品，专业 Python IDE
- **Sublime Text**：轻量级编辑器
- **Vim/Neovim**：命令行编辑器

### 包管理工具

- **pip**：Python 官方包管理器
  ```bash
  pip install package_name
  pip list  # 查看已安装的包
  ```

- **conda**：Anaconda 的包管理器，适合数据科学
  ```bash
  conda install package_name
  ```

### 虚拟环境

使用虚拟环境可以隔离不同项目的依赖：

**venv（Python 3.3+ 内置）**：
```bash
# 创建虚拟环境
python -m venv myenv

# Windows 激活
myenv\Scripts\activate

# macOS/Linux 激活
source myenv/bin/activate

# 退出虚拟环境
deactivate
```

## Python 代码风格

Python 社区遵循 **PEP 8** 代码风格指南，主要规则：

- 使用 4 个空格缩进（不使用 Tab）
- 每行代码不超过 79 个字符
- 使用有意义的变量名
- 函数和类之间空两行
- 导入语句放在文件顶部

示例：
```python
# 好的代码风格
def calculate_sum(a, b):
    """计算两个数的和"""
    return a + b


class Calculator:
    """计算器类"""
    pass
```
