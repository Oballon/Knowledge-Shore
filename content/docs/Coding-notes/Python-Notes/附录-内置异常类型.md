# Python 所有内置异常类型完整列表

## BaseException

- `SystemExit` - 解释器请求退出
- `KeyboardInterrupt` - 用户中断执行
- `GeneratorExit` - 生成器异常

- `ImportError` - 导入失败
- `ModuleNotFoundError` - 模块未找到

- `MemoryError` - 内存不足

- `NameError` - 名称未定义

- `IsADirectoryError` - 是目录

- `PermissionError` - 权限不足
- `ProcessLookupError` - 进程不存在
- `TimeoutError` - 操作超时
- `WindowsError` - Windows 特定错误

### ReferenceError

- `ReferenceError` - 弱引用错误

### RuntimeError

- `RuntimeError` - 运行时错误
- `NotImplementedError` - 未实现功能
- `RecursionError` - 递归过深

### SyntaxError

- `SyntaxError` - 语法错误
- `IndentationError` - 缩进错误

- `SystemError` - 解释器内部错误

### TypeError

- `TypeError` - 类型错误
