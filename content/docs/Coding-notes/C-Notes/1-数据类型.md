## 结构体

``` C
"基本语法"
struct Books {
    int data;
   ...
}book1={value1,value2,value3};

struct {
    int data;
   ...
}book1;
book1.data=value;

"book1：原地新建的实例（也可后续新建），可以有多个, 也可原地初始化"
"Books：结构体的名称，可以是任意的标识符"
"若Books省略，必须有原地实例book1"
```

### 访问成员方式

```
变量访问：book.data，传参printbook(book1)，定义void printbook(struct Books book)
指针访问：book->data，传参printbook(&book1)，定义void printbook(struct Books* book)
```

## 共用体

共用体所有成员共享同一段内存空间，其大小等于最大成员大小，任意时刻，共用体只能存放其中一个成员的值

``` C
union Data{
    int a;
    float b;
    char c;
}data; //data实例

typedef union unionData{
    int a;
    float b;
    char c;
}Data; //Data类型

"共用体与结构体 类型定义方式、访问方式完全一样(data.a  data->a)"
```

## 枚举类型

枚举类型通常用于为程序中的一组相关的常量取名字，以便于程序的可读性和维护性
``` C
enum DAY {
    MON,TUE=2,WED,THU,FRI,SAT,SUN
}day; //day为实例

typed enum DAY{
    MON=1,THE,WED,THU,FRI,SAT,SUN
}Day; //DAY为枚举类型

day=WED; // day=3
printf("%d",THE); //打印出2

"成员中部分赋值，则按次序填充；无赋值，则从零开始"
"枚举类型与结构体 类型定义方式完全一样，但为常量，无需访问修改成员变量之说"
```
